# Cubic Spline Interpolation Project

This is a project we're making for our Math 4103 course.

## Getting Started

Clone project and enter in folder:

```
cd vue-bootstrap-boilerplate
```

Run npm install:

```
npm i
```

Enjoy:

```
npm run serve
```
